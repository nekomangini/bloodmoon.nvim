# Bloodmoon.nvim

A Neovim colorscheme inspired by Shalltear Bloodfallen from Overlord

# Installation

Lazy.nvim

```lua
{
  "nekomangini/bloodmoon.nvim",
  lazy = false,
  priority = 1000,
}
```
